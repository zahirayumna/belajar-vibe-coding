# belajar-vibe-coding

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.14. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

## API Documentation

Dokumentasi interaktif (Swagger UI) tersedia di `http://localhost:3000/swagger` saat server berjalan (`bun run dev`). Semua endpoint `/api/users/*` bisa dicoba langsung dari situ lewat tombol "Try it out".
